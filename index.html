Uitvoer
Onderwerp
Notice and Takedown request [VOER DOMEINNAAM IN]

Inhoud
Dear Sir/Madam,

We’ve received several complaints regarding the website [VOER DOMEINNAAM IN].

We have reason to believe that this website is being used for fraudulent activities. Pending further investigation into this internet fraud we would like to request suspension of the hosting provided for the domain [VOER DOMEINNAAM IN].

This was revealed by:
  • Reports have been made to the police requesting criminal prosecution
  • The website doesn’t have a Dutch Chamber of Commerce number
  • The website doesn’t have a Dutch VAT identification number
  • The website doesn’t have valid address data
  • The website abuses a Dutch Chamber of Commerce number
  • The website abuses a Dutch VAT identification number
  • The website abuses a web store certification
  • The website abuses address data
  • The website collects payments, but does not deliver products
  • The website sells products which are priced below market value
  • The website only offers payment by manual bank transfer
  • The website only offers pay by crypto currency
  • The website used multiple languages mixed together
  • The website is linkable to other malicious websites
  • The domain name is registered very recently
  • The whois data of the domain name is abused
  • Customer service does not respond to customer requests
We would like to refer to the ‘Notice and Takedown’ code as agreed between internet service- and hosting providers.

We already put the website [VOER DOMEINNAAM IN] on the list of fraudulent websites on the official website www.politie.nl.

This message has been sent under the authority of the Dutch police and the public prosecutor of the Noord-Holland district prosecutor's office.

Regards,

National Reporting Center Internetfraud


Uitvoer
Onderwerp
Zekerheid grenzende waarschijnlijkheid malafide website [VOER DOMEINNAAM IN]

Inhoud
Beste,

Het LMIO heeft meldingen ontvangen over de website:

[VOER DOMEINNAAM IN]

Uit controle van de website is gebleken dat we met een aan zekerheid grenzende waarschijnlijkheid kunnen spreken over een malafide website.

Dat is gebleken uit:
  • Er zijn aangiften gedaan bij de politie met het verzoek tot strafrechtelijke vervolging
  • De website toont geen KvK-nummer, wat wel wettelijk verplicht is
  • De website vermeldt geen btw-identificatienummer, wat wel geadviseerd wordt door ACM
  • De website heeft geen geldig bezoekadres, vestigingsadres of postadres
  • De website misbruikt een KvK-nummer
  • De website misbruikt een btw-identificatienummer
  • De website misbruikt een webwinkelkeurmerk
  • De website misbruikt een bezoekadres, vestigingsadres of postadres
  • Uit aangiftes komt naar voren dat klanten producten na betaling niet geleverd krijgen
  • De website verkoopt producten die onder de marktwaarde zijn geprijsd, te mooi om waar te zijn
  • De website biedt alleen betalen per handmatige bankoverschrijving, dit is erg ongebruikelijk
  • De website biedt alleen betalen per crypto valuta, dit is erg ongebruikelijk
  • De website gebruikte meerdere talen door elkaar, terwijl de doelgroep Nederland is
  • De website is te linken aan andere malafide websites
  • De website is zeer recent geregistreerd, maar suggereert aan klanten al langer te bestaan
  • De whois-gegevens van de domeinnaam worden misbruikt
  • De klantenservice reageert niet op vragen of opmerkingen vanuit klanten
De website staat inmiddels vermeld in de lijst met malafide handelspartijen op de website www.politie.nl.

Daarnaast is er een verzoek verzonden naar de host om passende maatregelen te nemen tegen de website.

Help
Uitleg
  1. Vul een domeinnaam in, zonder https://www.
  2. Kies uit Opties hetgeen wat van toepassing is
  3. Druk op Bekijk SIP-request, Bekijk NTD-request of Bekijk partner-mail
  4. Druk op Nieuwe e-mail om een lege e-mail te openen
  5. Druk op Kopieer onderwerp en/of Kopieer inhoud
  6. Plak de gekopieerde tekst in de lege e-mail
  7. Druk op Herstel waarden om opnieuw te beginnen
